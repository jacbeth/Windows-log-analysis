SOC Analyst Labs

endpoint detection, log analysis, and incident investigation.

* Windows 11
* Sysmon
* VirtualBox
* PowerShell
* Event Viewer

Labs

1. Suspicious PowerShell Execution
2. Phishing Email Analysis
3. Malware Behavioral Analysis
4. Brute Force Detection
5. Lateral Movement Detection
